# Mandelbrot
Mandelbrot fractal implemented with OpenGL Shaders

## Compiling
```bash
mkdir build
cd bulid
cmake ..
make
./Fractals
```
Enjoy!

## Usage
Scroll to zoom in and out

Drag the canvas to move the fractal (fractal does *not* update in realtime)

'r' to reset to default zoom
